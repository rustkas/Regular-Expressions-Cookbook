basic
=====

An OTP library
## Create new project

----	
	
	# all commands in one string
	$ rebar3 new lib basic && cd basic
	
## Get dependencies
	$ rebar3 deps	

## Format
	$ rebar3 format
	
## EUnit
-----
	$ rebar3 eunit
	$ rebar3 eunit -m basic_tests

An OTP library

Build
-----

    $ rebar3 compile
